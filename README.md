# chatbot_husc
 AI-powered academic Q&amp;A chatbot for Hue University of Sciences (HUSC). Built with Flask and FastAPI, featuring a hybrid RAG pipeline (BGE-M3, FAISS/Qdrant, BM25, cross-encoder reranking) powered by Google Gemini. Includes structure-aware PDF chunking, anti-hallucination citation validation, intent classification, and rate limiting.
